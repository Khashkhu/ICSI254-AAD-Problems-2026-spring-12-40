Атом

Нэг төрлийн атом нэгж радиустай бөмбөрцөг эзлэн оршдог. Түүний N ширхэг электрон нь бөмбөрцгийн гадаргуу болох бөмбөлөг дээр ялгаатай байрлалд байх бөгөөд аль ч 4 нь нэг хавтгайд байрладаггүй. 
Бид ийм атомыг 2 хэсэгт хуваах даалгавартай. Хуваахдаа дурын гурван электроныг дайрсан хавтгайгаар зүсэх бөгөөд энэ гурван электрон устаж атом 2 хэсэг болно.
Тэгвэл ядаж нэг хэсэгт нь яг M ширхэг электрон байхаар хэдэн янзаар хуваах боломжтой вэ?

Оролт:

Гаралт:
