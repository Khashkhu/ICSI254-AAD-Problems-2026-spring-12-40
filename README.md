## Бодлого: Хатан

## Товч тайлбар
Өрөөнүүдийг хонгилоор холбосон граф өгөгдөнө. Боржигчид 1-р өрөөнөөс
эхлэн n-р өрөөнд хүрэхийг оролдох ба өрөө бүрт байгаа мэрэгчдийн тооноос
хамааран тулалдаан явагдана. Тулалдааны дүрмийн дагуу амьд үлдэх
боржигчдын тоо өөрчлөгдөнө.

Зорилго нь 1-р өрөөнөөс n-р өрөө хүртэл очиж хамгийн ихдээ хэдэн
боржигчин амьд хүрч чадахыг олох явдал юм.

 Бодлогын бүрэн өгүүлбэр, оролт/гаралт, жишээ: `Uguulber.md`

## Шийдлийн санаа 
- Өрөөнүүд болон хонгилуудыг граф гэж үзнэ.
- Боржигчид зам дагуух бүх өрөөнд тулалдаанд оролцоно.
- Өрөө бүрт:
  - Хэрэв боржигчдын тоо > мэрэгчдийн тоо → боржигчид бүгд амьд үлдэнэ
  - Эсрэг тохиолдолд -> x/2 боржигчин амьд үлдэнэ
- 1 болон n-р өрөөнд мөн тулалдаан явагдана.
- Граф дээр 1-ээс n хүртэлх бүх боломжит замуудыг авч үзэж,
  амьд үлдэх боржигчдын тоог тооцоод хамгийн ихийг нь сонгоно.


## Бодлогын өгүүлбэр

Мэргэд аймгаас хатнаа буцаан авахын тулд Тэмүжиний толгойлсон боржигин овгийн x тооны цэргүүд нэгэн ууланд иржээ. Мэргэдүүд ууланд n ширхэг өрөөг үүсгэсэн ба тэдгээр нь хоорондоо аль ч чиглэлд явж болох m ширхэг хонгилоор холбогдсон. Боржигинчууд эхлээд 1-р өрөөнд орж ирэх ба хатан n-р өрөөнд байгаа.

i-р өрөөнд aᵢ тооны мэргэдүүд байрлана. Боржигинчууд орж ирэхэд бүрт тулалдаан эхэлнэ.
Хэрэв боржигинчуудын тоо мэргэдүүдийн тооноос их байвал бүх мэрэгдүүд алагдаж, боржигин цэргүүд бүгд амьд үлддэг. Эсрэг тохиолдолд бүх мэргэдүүд алагдах боловч боржигинчуудын  тал нь амь үрэгддэг.Хэрэв тулааны эхэнд х тооны боржигинчууд байсан бол x/2 (доод бүхэл тоо) нь амьд үлдэнэ. Боржигинчууд  очсон өрөө бүртээ мэргэдүүдтэй тулалдах ба 1-р болон 
n-р өрөөнд мөн тулалдана.

1-р өрөөнөөс n-р өрөө хүртэл хамгийн ихдээ хэдэн боржигинчууд очиж чадахыг ол.

## Оролт

- Эхний мөрөнд мэргэдүүдийн өрөөний тоо болон хонгилуудын тоо болох n,m бүхэл тоонууд өгөгдөнө (1 ≤ n, m ≤ 2 · 10⁵)

- Дараагийн m тооны мөрөнд хонгилуудыг тодорхойлно. i-р хонгил нь uᵢ ба vᵢ тоонуудаар тодорхойлогдох ба тэдгээр нь уг хонгилоор хоорондоо холбогдож буй өрөөнүүдийн дугаар юм.
(1 ≤ uᵢ, vᵢ ≤ n, uᵢ ≠ vᵢ) Хос өрөө бүр хоорондоо нэгээс олонгүй хонгилоор холбогдох нь баталгаатай.

- Дараагийн мөрөнд x тоо байх ба энэ нь 1-р өрөө рүү орж ирж буй боржигинчуудын тоо.
  (1 ≤ x ≤ 10⁹)

- Сүүлийн мөрөнд a₁, a₂, …, aₙ тоонууд байрлах ба эдгээр нь өрөөнүүд дэх мэргэдүүдийн тоо юм. (1 ≤ aᵢ ≤ 10⁹)

## Гаралт

1-р өрөөнөөс эхлээд n-р өрөөнд очиж чадах боржигинчуудын хамгийн их тоог гаргана. Ядаж нэг боржигон хүн амьд үлдэх нэг ч аялал олдохгүй бол 0-ийг гаргана.
