# ICSI254-AAD-Problems-2026-spring-12-40
Хоёр тоон дарааллын давтамжийн ялгаатай байдлыг судлахад туслана уу. N урттай A ба В дарааллын ялгаатай байдал нь дараах томъёогоор илэрхийлэгдэнэ. 
sum |Ai - Bi|
Хоёр дарааллыг харьцуулахын өмнө дараах зүйлийг хийх хэрэгтэй:
    1. эхлээд Х бүхэл тоог сонгон авч эхний дарааллын элемэнт бүр дээр нэмэх замаар дарааллыг өөрчлөх.
    2. дараа нь мөн эхний дарааллын К-аас хэтрэхгүй тооны дурын элемэнтүүдийн утгыг өөрчлөх.
    
Тодруулбал, Х бүхэл тоо сонгон эхний дарааллын утгыг өөрчилж, К тооны зарим элемэнтийн утгыг өөрчилснөөр хоёр дарааллын давтамжийн ялгаатай байдал хамгийн бага байлгах ёстой.

Оролт:
Эхний мөрөнд N ба K тоонууд байна.
Хоёр дахь мөрөнд N тооны Ai дарааллын бүхэл тоон утгууд
Гурав дахь мөрөнд N тооны Вi дарааллын бүхэл тоон утгууд

нэг мөрөнд хоёр дарааллын байж болох хамгийн бага ялгаатай байдлыг хэвлэнэ.
